CMPS.SharedKernel
Цей проект є найглибшим шаром, який не залежить ні від чого іншого (крім стандартних бібліотек .NET). Він містить базові абстракції, які використовуються по всьому додатку.

CMPS.SharedKernel/
├── CMPS.SharedKernel.csproj
├── Abstractions/
│   ├── Entity.cs            // Базовий клас для доменних сутностей
│   ├── AggregateRoot.cs     // Базовий клас для коренів агрегатів
│   ├── ValueObject.cs       // Базовий клас для об'єктів-значень
│   ├── IDomainEvent.cs      // Інтерфейс для доменних подій
│   └── IRepository.cs       // (Опціонально) Базовий інтерфейс для всіх репозиторіїв
├── Events/
│   └── IDomainEventDispatcher.cs // Інтерфейс для диспетчера доменних подій
├── Exceptions/
│   └── DomainException.cs   // Базовий клас для доменних винятків
├── Helpers/
│   └── Guard.cs             // Утилітарний клас для перевірок аргументів
├── Specifications/
│   └── ISpecification.cs    // Інтерфейс для шаблону специфікації (для фільтрації, валідації тощо)
└── Utilities/
    └── DateService.cs       // (Приклад) Загальний сервіс для роботи з датами
Пояснення:

Abstractions: Містить фундаментальні будівельні блоки DDD. Ці класи та інтерфейси слугуватимуть основою для всіх сутностей, 
об'єктів-значень та агрегатів у ваших Bounded Contexts.

Events: Інтерфейс для диспетчеризації доменних подій.

Exceptions: Загальні винятки доменного рівня.

Helpers, Utilities: Дрібні загальні утиліти, які не належать до жодного конкретного домену.

Specifications: Допомагає інкапсулювати бізнес-правила фільтрації або валідації.
